# صامل (Samel) — منصة الإعلانات المبوبة والمزادات السعودية

سوق سعودي عربي (RTL) يجمع الإعلانات المبوبة مع نظام مزادات مباشر، ونظام مصداقية متبادل يحاسب الطرفين بعد كل معاملة.

## التشغيل

```bash
npm install
npm run gen:images   # توليد الصور التجريبية (مرة واحدة)
npm run db:push      # إنشاء قاعدة البيانات SQLite
npm run db:seed      # تعبئة بيانات تجريبية
npm run dev          # http://localhost:3000
```

## التقنيات

- **Next.js 16** (App Router, Turbopack) + TypeScript strict + React 19
- **Tailwind CSS v4** مع توكنز التصميم (برتقالي/أحمر/أسود/أبيض) وخطوط Cairo + IBM Plex Sans Arabic
- **Prisma 6 + SQLite** للتطوير (قابلة للتبديل إلى PostgreSQL بتغيير الـ provider)
- جلسات JWT (jose) في كوكيز HttpOnly + bcrypt

## الميزات الرئيسية

### المزادات (الميزة التنافسية)
- مزايدة لحظية (polling كل 3 ثوانٍ — قابلة للترقية إلى WebSocket)
- هويات مقنّعة للمزايدين (`صامل_XXX`) — الهويات الحقيقية تظهر فقط في سجل المزايدات بلوحة الأدمن
- حماية من القنص: مزايدة في آخر دقيقتين تمدد المزاد دقيقتين
- منع البائع من المزايدة على مزاده + حد أدنى للزيادة + شراء فوري اختياري
- إنهاء تلقائي كسول (lazy finalization) مع إشعارات للفائز والبائع وكشف بيانات التواصل

### نظام المصداقية والتحقق المتبادل
- نافذة تأكيد 48 ساعة بعد كل صفقة (تسليم/استلام)
- مصفوفة النقاط: تأكيد متبادل +5 · تجاهل -3/-5 · خسارة نزاع -15
- نزاعات مع إفادات ومراجعة من فريق الدعم + تقييم متبادل (نجوم) بعد التأكيد

### السوق
- تسجيل بالبريد الإلكتروني (الجوال اختياري في الإعدادات، مهيأ للتوثيق بـOTP مستقبلاً)
- بحث عربي ذكي: تطبيع الهمزات والتشكيل والتاء المربوطة (`ايفون` = `آيفون`) + اقتراحات فورية + ترتيب بالصلة
- شات داخلي + تعليقات على الإعلانات + بلاغات على (إعلان/مستخدم/تعليق/رسالة)
- متاجر شخصية برابط خاص + نقاط نشاط يومية تُستبدل بتمييز الإعلانات
- رفع صور حتى 5MB مع ضغط تلقائي في المتصفح للصور الأكبر
- 43 فئة سعودية + فلاتر (مدينة/سعر/حالة/نوع) + طرق تسليم

### لوحة الأدمن (`/admin`)
صلاحيات RBAC (مدير/مشرف/دعم/محاسب): إحصائيات، مستخدمون (حظر/نقاط)،
إعلانات (تمييز/حذف)، سجل مزايدات كامل، نزاعات، بلاغات، بانرات (صور أو
أكواد مضمّنة AdSense/يوتيوب)، باقات وأسعار قابلة للتعديل، كلمات محظورة،
إشعار عام لكل المستخدمين، وسجل تدقيق (Audit Log).

## ملاحظات معمارية

- `src/lib/arabic.ts` — التطبيع العربي يطبق نفسه عند الفهرسة (`Listing.searchText`) وعند البحث
- `src/lib/credibility.ts` — مصفوفة التحقق المتبادل وقرارات النزاعات
- `src/lib/auction.ts` — إنهاء المزادات المنتهية يُستدعى من مسارات القراءة (بديل cron في التطوير)
- المدفوعات (Moyasar/Tap) وOTP (Taqnyat) وWebSocket نقاط ربط مؤجلة للإنتاج — الواجهات جاهزة لها
